#### 分屏
1. 水平分屏
```vim
:split 文件名
```
快捷键：
```vim
<C-w> s 或 :sp
```
说明：将当前窗口一分为二，上下排列。
2. 垂直分屏
```vim
:vsplit 文件名
```
快捷键：
```vim
<C-w> v 或 :vsp
```
说明：左右并排显式两个窗口
3. 窗口切换
```vim
<C-w> <方向键>

<C-w> h : 切换到左边窗口
<C-w> l : 切换到右边窗口
<C-w> j : 切换到下边窗口
<C-w> k : 切换到上边窗口
```

4. 调整窗口大小
- `<C-w> +`：增加当前窗口高度
- `<C-w> -`：减少当前窗口高度
- `<C-w> >`：增加当前窗口宽度
- `<C-w> <`：减少当前窗口宽度
